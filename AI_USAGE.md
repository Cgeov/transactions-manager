# Uso de Inteligencia Artificial

Durante el desarrollo de **Gestor de Transacciones** se utilizó **Antigravity (Google)** como apoyo para tareas de desarrollo, refactorización y optimización de código.

## ¿Cómo se utilizó?

La herramienta se empleó principalmente para:

- Proponer mejoras en el manejo de suscripciones y ciclos de vida de Angular.
- Refactorizar y simplificar componentes y layouts.
- Limpiar código generado automáticamente y comentarios innecesarios.
- Apoyar la generación de datos de prueba y scripts auxiliares.

## Revisión del código generado

Todo el código sugerido fue revisado y validado manualmente antes de incorporarse al proyecto. Se realizaron:

- Compilaciones y validaciones de TypeScript.
- Pruebas funcionales de filtros, búsquedas y reprocesos.
- Ajustes y correcciones sobre las propuestas iniciales cuando fue necesario.

## Decisiones del desarrollador

Las decisiones de arquitectura, diseño, experiencia de usuario e implementación final fueron tomadas por mi persona. La IA se utilizó como herramienta de asistencia, mientras que la integración, validación y adaptación del código fueron realizadas manualmente.

## Uso de la IA
Para evitar incorporar código innecesario o inseguro:

- Se revisó y comprendió cada cambio antes de integrarlo.
- Se descartaron soluciones que añadían dependencias externas o complejidad innecesaria.
- Se mantuvo el tipado estricto de TypeScript.
- Se validó el comportamiento mediante pruebas manuales y revisión de código.
- No se incorporó código cuyo funcionamiento no pudiera explicarse o mantenerse posteriormente.
